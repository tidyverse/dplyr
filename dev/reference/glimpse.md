# Get a glimpse of your data

`glimpse()` is like a transposed version of
[`print()`](https://rdrr.io/r/base/print.html): columns run down the
page, and data runs across. This makes it possible to see every column
in a data frame. It's a little like
[`str()`](https://rdrr.io/r/utils/str.html) applied to a data frame but
it tries to show you as much data as possible. (And it always shows the
underlying data, even when applied to a remote data source.)

`glimpse()` is provided by the pillar package, and re-exported by dplyr.
See
[`pillar::glimpse()`](https://pillar.r-lib.org/reference/glimpse.html)
for more details.

## Value

x original x is (invisibly) returned, allowing `glimpse()` to be used
within a data pipeline.
