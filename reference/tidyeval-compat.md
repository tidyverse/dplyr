# Other tidy eval tools

These tidy eval functions are no longer for normal usage, but are still
exported from dplyr for backward compatibility. See
[`?rlang::args_data_masking`](https://rlang.r-lib.org/reference/args_data_masking.html)
and
[`vignette("programming")`](https://dplyr.tidyverse.org/articles/programming.md)
for the latest recommendations.

- [expr()](https://rlang.r-lib.org/reference/expr.html)

- [enquo()](https://rlang.r-lib.org/reference/enquo.html)

- [enquos()](https://rlang.r-lib.org/reference/enquo.html)

- [sym()](https://rlang.r-lib.org/reference/sym.html)

- [syms()](https://rlang.r-lib.org/reference/sym.html)

- [as_label()](https://rlang.r-lib.org/reference/as_label.html)

- [quo()](https://rlang.r-lib.org/reference/defusing-advanced.html)

- [quos()](https://rlang.r-lib.org/reference/defusing-advanced.html)

- [quo_name()](https://rlang.r-lib.org/reference/quo_label.html)

- [ensym()](https://rlang.r-lib.org/reference/defusing-advanced.html)

- [ensyms()](https://rlang.r-lib.org/reference/defusing-advanced.html)

- [enexpr()](https://rlang.r-lib.org/reference/defusing-advanced.html)

- [enexprs()](https://rlang.r-lib.org/reference/defusing-advanced.html)
